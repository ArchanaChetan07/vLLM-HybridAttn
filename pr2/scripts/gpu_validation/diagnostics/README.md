# GPU diagnostics (Stage 0 parity bisect)

Scripts in this directory are **development probes** for HF vs vLLM bisect on real GPU
hardware. They are not part of the gated validation suite and are not required for CI.

**Gated suite:** `../run_all_gpu_validation.sh` (Steps 0-4, 6, B).

Do not cite diagnostic output as merge evidence unless results are copied into
[docs/VALIDATION_REPORT.md](../../../docs/VALIDATION_REPORT.md) with provenance.
